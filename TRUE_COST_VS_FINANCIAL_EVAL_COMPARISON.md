### **2. الحسابات والصيغ**

#### **تقرير التكلفة الحقيقية:**

```javascript
// 1. حساب التكلفة الديناميكية
calculatedConstructionCost = dynamicBUA × dynamicPricePerSqft

// 2. حساب أتعاب التصميم (ديناميكي)
if (designMethod === 'PERCENTAGE') {
  quotedDesignFee = (designPct / 100) × calculatedConstructionCost
} else {
  quotedDesignFee = designAmount  // Lump Sum
}

// 3. حساب أتعاب الإشراف (ديناميكي)
if (supervisionMethod === 'PERCENTAGE') {
  quotedSupervisionFee = (supervisionPct / 100) × calculatedConstructionCost
} else if (supervisionMethod === 'MONTHLY_RATE') {
  quotedSupervisionFee = monthlyRate × dynamicDuration
} else {
  quotedSupervisionFee = supervisionAmount  // Lump Sum
}

// 4. التكلفة الحقيقية
trueDesignFee = quotedDesignFee + designScopeGap
adjustedSupervisionFee = quotedSupervisionFee + supervisionGap
totalTrueCost = trueDesignFee + adjustedSupervisionFee

// 5. الترتيب والسكور
lowestCost = min(totalTrueCost for all consultants)
rank = sorted by totalTrueCost ascending
score = (lowestCost / totalTrueCost) × 100
```

**المميزات:**
- ✅ حسابات **ديناميكية** - تتحدث عند تغيير المتغيرات
- ✅ **تجاوزات يدوية** - يمكن تعديل أي خلية
- ✅ **معادلات واضحة** - كل حساب معروض

---

#### **التقييم المالي في مركز القيادة:**

```javascript
// 1. حساب التكلفة الثابتة
constructionCost = project.bua × project.pricePerSqft

// 2. حساب أتعاب التصميم (ثابتة)
if (designType === 'pct') {
  designAmount = constructionCost × (designValue / 100)
} else {
  designAmount = designValue  // Lump Sum
}

// 3. حساب أتعاب الإشراف (ثابتة)
if (supervisionType === 'pct') {
  supervisionAmount = constructionCost × (supervisionValue / 100)
} else {
  supervisionAmount = supervisionValue  // Lump Sum
}

// 4. فجوات النطاق
designGapCost = fin.designGapOverride || cpaDesignGap
supervisionGapCost = fin.supervisionGapOverride || cpaSupervisionGap

// 5. المجموع
designTotal = designAmount + designGapCost
supervisionTotal = supervisionAmount + supervisionGapCost
totalFees = designTotal + supervisionTotal

// 6. السكور المالي
lowestFee = min(totalFees for all consultants)
financialScore = (lowestFee / totalFees) × 100
```

**المميزات:**
- ✅ حسابات **ثابتة** - لا تتغير عند التحديث
- ✅ **تجاوزات يدوية** - محفوظة في قاعدة البيانات
- ✅ **مقارنة بالمتوسط** - نسبة الانحراف عن المتوسط

## 🚨 المشاكل المكتشفة

### **1. عدم التطابق:**
- التقرير يحسب بشكل **ديناميكي**
- التقييم يحسب بشكل **ثابت**
- **النتيجة:** قد تختلف الأرقام!

### **2. مصادر البيانات مختلفة:**
- التقرير يستخدم `cpa_evaluation_results`
- التقييم يستخدم `financial_data` + `cpa_evaluation_results`
- **النتيجة:** قد تكون البيانات غير متطابقة!

### **3. عدم وجود اعتماد في التقييم:**
- التقرير له نظام اعتماد
- التقييم بدون اعتماد
- **النتيجة:** قد يتم تعديل البيانات بدون تتبع!

## 💡 التوصيات

### **الحل الأمثل:**
جعل **التقييم المالي في مركز القيادة** نسخة طبق الأصل من **تقرير التكلفة الحقيقية**:

1. **استخدام نفس البيانات** من `cpa_evaluation_results`
